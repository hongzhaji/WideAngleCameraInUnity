# WideAngleCameraInUnity
在unity中打开广角摄像头（仅仅支持mi9）
# ------------ unity -------------------
## Assert:
   ### 1.创建自己的Unity项目，然后拷贝Assert 到unity项目
   ### 2.设置为Android编译
  
  
# -----------  Android -------------
## libs : Android 工程需要的库文件
## src ： Android 工程源文件 （包含jni 和 renderscript）

  ### 1. 新建Android工程
  ### 2.替换src 和lib 
  ### 3.编译
  ### 4. 替换 unity项目中的aar
